LCG


This course will be taught at the Undergraduate Program on Genomic Sciences (LCG in Spanish) from the National Autonomous University of Mexico (UNAM in Spanish). It is directed at the third semester students from the sixth generation. It is meant as a Bioconductor overview course and its official name is Seminar III. It has an official length of 32 class hours. For more information on this undergraduate program visit LCG's main page. For example, the sections on Development of the Genomic Sciences in Mexico and LCG's International Evaluation could be of your interest.

lectures


In this section you can find a small description for every lecture. Additionally, you can download the lectures in PDF format, the R code and/or the .Rnw file to make the lecture files.

Class 01: review  [code]  [.Rnw] 
The first class of the semester! We'll review R, how to make plots and learn how to use the apply functions.

Class 02: bioconductor and documentation  [code]  [.Rnw] 
On this class, we'll introduce you to Bioconductor, learn to make a basic heatmap and foment reproducible research through vignette files using Sweave.

Class 03: advanced plotting  [code]  [.Rnw] 
We'll learn how to make advanced plots using the lattice, plotrix, ggplots2 and car packages.

Class 04: public data  [code]  [.Rnw] 
We'll learn how to download public data using biomaRt, GEOquery, ArrayExpress, annotate and KEGGSOAP.
