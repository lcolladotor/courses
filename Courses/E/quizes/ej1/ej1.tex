\begin{enumerate}
  \item Empiezen por hacer el ejercicio simple que les pusimos en la clase. \emph{Entendemos que el de los números de Fibonnaci es más complicado}.
  \bigskip
  
  \item Abran \texttt{R} y utilicen los datos \texttt{2 5 4 10 8} para:
  \begin{enumerate}
    \item Almacenarlos en un vector de datos x.
	\item Encuentren el cuadrado de cada número.
	\item Substraigan 3 de cada número.
	\item Substraigan 5 de cada número y luego encuentren su raíz.
  \end{enumerate}
  Hagan esto usando funciones (aprovéchense del reciclaje).
  \bigskip
  
  \item Siendo \texttt{x} \& \texttt{y}:
\begin{Schunk}
\begin{Sinput}
> x <- c(1, 3, 5, 7, 9)
> y <- c(2, 3, 5, 7, 11, 13)
\end{Sinput}
\end{Schunk}
  Sin correr los comandos, cual es el resultado de:
  \begin{enumerate}
    \item x+1
    \item y*2
	\item length(x) y length(y)
	\item x + y (hay reciclaje)
	\item sum(x>5) y sum(x[x>5])
	\item sum(x>5 | x<3)
	\item y[3]
	\item y[-3]
	\item y[x] (\textquestiondown{}Qué es NA?)
	\item y[y>=7]
  \end{enumerate}
  \bigskip
  
\end{enumerate}
